Country Statistics API





API Information:

This API displays a list of data respective to the selected country. It includes its image, its name, the region it's located in, its population,the main spoken language, and the currency used.

Where this API was found: API Documentation

Options used for the API request:
I created an XMLHttpRequest object, opened the JSON file holding the country's data with a GET request, and extracted information when my request was validated.

Application for this API:
1. To be used for a flight or booking website to display general but important information about the country the user wants to travel to.
2. To be used by geography students for a quick overview of general but important data and demographics of a country.
